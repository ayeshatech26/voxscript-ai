# 🎙️ VoxScript AI - Professional Speech Studio

VoxScript AI is a high-performance, real-time voice-to-text transcription tool. Specially designed for bilingual accuracy, it supports both **English (US)** and **Urdu (Pakistan)** with a premium, animated user interface.

## ✨ Key Features
- **Bilingual Support:** High phonetic accuracy for Urdu and English.
- **Real-time Transcription:** Instant text-to-speech conversion using Web Speech API.
- **Premium UI:** Modern Glassmorphism design with sleek animations.
- **Privacy First:** No audio data is stored on servers; processing happens locally.
- **Utility Tools:** One-click Copy to Clipboard and Clear Canvas features.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Framer Motion
- **Icons:** Lucide-React
- **Processing:** Native Browser Web Speech Engine
